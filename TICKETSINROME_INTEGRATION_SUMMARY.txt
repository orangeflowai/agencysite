================================================================================

One-command deployment:

  HETZNER_IP=your.server.ip ./deploy-ticketsinrome-hetzner.sh

Or manual deployment:

  1. SSH to server: ssh root@your.server.ip
  2. Navigate: cd /var/www/ticketsinrome
  3. Pull code: git pull origin main
  4. Install: npm install --legacy-peer-deps
  5. Build: npm run build
  6. Restart: pm2 restart ticketsinrome
