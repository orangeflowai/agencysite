## Troubleshooting

### Tours not showing on website?
- Check that `tenant` is set to `goldenrometour`
- Verify `status` is `active`
- Ensure `category` is `vatican`
- Check Vercel environment variable `DATA_SOURCE=payload`

### Product pages showing 404?
- Verify the slug matches exactly (no typos)
- Check that the tour is published/active
- Clear Next.js cache by redeploying

### Images not loading?
- Ensure images are uploaded to Payload CMS
- Check that Cloudflare R2 is configured correctly
- Verify image URLs in the tour data
