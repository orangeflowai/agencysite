## ✨ ANTIGRAVITY-SPECIFIC IMPLEMENTATION NOTES

1. **Marquee bar** → Use a native Marquee element or Ticker component, infinite loop
2. **Hero overlapping search bar** → Set hero to `position: relative`, search bar `position: absolute; bottom: -40px; z-index: 10`
3. **Floating cards on hero** → Absolutely positioned inside the grid cell, `transform: rotate(-3deg)` for dynamism
4. **Arch image shapes** → `border-radius: 50% 50% 0 0 / 60% 60% 0 0`
5. **Section transitions** → Use diagonal clip-path: `clip-path: polygon(0 0, 100% 0, 100% 90%, 0 100%)` on section bottoms
6. **Stats bar** → `overflow: hidden` with a subtle sliding animation on scroll
7. **Card hover** → `transform: translateY(-8px)`, shadow deepens, gold border appears
8. **Gold accents** → Use consistently at 20% of visual weight (not dominant)
9. **Marble texture** → SVG noise filter or CSS `background-image` with low-opacity marble PNG
10. **Parallax** → On hero + CTA banner sections, `background-attachment: fixed`
